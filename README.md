AUTOMATED CONTAINER DEPLOYMENT AND ADMINISTRATION IN THE CLOUD  

Once developed, this project demonstrates the containerisation of a Python (Flask) application and deployment to AWS EC2. The infrastructure is provisioned using AWS CloudFormation (IaC) while Docker provides a snug fit between the runtime on local development (macOS) and EC2. 

Features List
    =>Python Flask microservices
    =>Dockerised Application
    =>AWS EC2 Deployment
    =>Security Group Rules for Port 22 + 8080
    =>CloudFormation Template ready for reproducible Infrastructure as Code
    =>GitHub Actions workflow for automated build & deploy
